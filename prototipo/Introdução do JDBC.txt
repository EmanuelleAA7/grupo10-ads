O Que é o JDBC?
É um conjunto de classes que fazem aplicações para se conectar ao bancos de dados relacionais; 

Quais são os três componentes princiapis? 

Connection -- é utilizado para fazer as conexões com o banco de dados, sem ele é impossivel fazer conexões com o banco de dados;

Statement --- é utilizado para fazer os comando em SQL como o select ele também utiliza o connection;

ResultSet --- é utilizado para ler todos os resultados que os statement trás; 

O que é o drive ?

os drives é uma implemetação de um objeto espeficico de um banco de dados, ou seja se precisarmos acessar um banco de dados oracle temos que ir 
atras de um JDBC ORACLE!
